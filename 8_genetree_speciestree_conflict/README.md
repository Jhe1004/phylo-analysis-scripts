# 8_genetree_speciestree_conflict

## 功能概述
这一部分用于分析 gene tree 与 species tree 的冲突关系，并通过定年和 densitree 可视化进一步解释冲突模式。

## 子目录说明
- `0_extract_subtree`：提取关注类群的子树
- `1_generate_treePL_configfile`：生成 treePL 配置
- `2_treePL`：批量运行 treePL
- `3_draw_densitree`：绘制 densitree 和节点支持图

## 使用建议
请优先进入具体子目录阅读对应的 `README.md`，再修改脚本顶部配置并运行。
