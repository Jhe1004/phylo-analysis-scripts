# 20_Chloroplast

## 功能概述
这一部分集中放置叶绿体相关分析脚本，包括组装、注释整理、SSR、密码子偏好、高变区分析以及多种建树流程。

## 子目录说明
- `0_密码子偏好`：密码子使用偏好统计
- `1_ssr分析`：SSR 检测
- `2_基因正选择分析`：正选择分析准备
- `3_基因表格生成`：基因组成统计表
- `4_高变区`：Pi 统计和高变区定位
- `5_mvista`：mVISTA 相关输入整理
- `6_gb序列重命名`：统一重命名 gb / 位点
- `7_chloroplast组装`：叶绿体组装
- `8_叶绿体用cds建树`：抽取叶绿体同源 CDS 建树
- `9_复杂基因结构建树`：复杂结构叶绿体建树流程
- `PGA`：保留的上游注释工具目录

## 使用建议
请优先进入具体子目录阅读对应的 `README.md`，再修改脚本顶部配置并运行。
