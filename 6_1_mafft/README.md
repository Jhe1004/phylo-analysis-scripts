# 6_1_mafft

## 功能概述
用于批量执行 MAFFT 多序列比对。脚本可以并行处理多个基因文件，并支持为每个 MAFFT 任务指定内部线程数。

## 正式入口
- `mafft.py`

## 输入与输出
### 输入
- `input/` 中待比对的 FASTA 文件
### 输出
- `output/` 中的比对结果 FASTA

## 使用方式

1. 打开正式脚本，直接修改脚本最前面的配置区。
2. 把输入文件放入当前目录的 `input/`。
3. 在当前目录运行对应脚本。
4. 结果会统一写入 `output/`。

本项目已经统一约定：

- 不通过 CLI 交互式传参
- 路径尽量写相对路径
- 外部软件优先从 `dependencies/bin/`、指定 conda 环境、系统 `PATH` 查找

## 备注
- 支持多进程和多线程两级并行。
