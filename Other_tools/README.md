# Other_tools

## 功能概述

这个目录用于集中存放一些不属于主流程、但在实际项目中经常会用到的小工具脚本。它们大多是单独的小功能，例如树处理、统计、绘图、FASTA/FASTQ 操作和 ArcGIS 辅助处理。

## 主要类别

- `0_树相关`：树剪枝、重定根、提取物种名、树格式处理等
- `1_统计各种数字`：N50、gap 统计、位点统计等
- `2_画各种图`：直方图、小提琴图等
- `3_fasta&fastq相关`：矩阵抽取、重命名、bootstrap、文件拆分等
- `4_arcgis相关`：ArcGIS 数据处理辅助脚本
- `5_sra数据提交`：SRA / NGDC 数据提交辅助脚本

## 使用说明

这个目录不是一条统一大流程，建议直接进入具体子目录查看脚本名和目录内说明后使用。
