# 2_cdhit

## 功能概述
用于对转录本或蛋白序列做去冗余处理，减少后续 ORF 预测和直系同源分析中的冗余序列数量。脚本会批量处理输入目录中的 FASTA 文件，并统一输出聚类后的结果。

## 正式入口
- `cdhit.py`

## 输入与输出
### 输入
- `input/` 中待去冗余的 FASTA 文件
### 输出
- `output/` 中 CD-HIT 聚类结果

## 使用方式

1. 打开正式脚本，直接修改脚本最前面的配置区。
2. 把输入文件放入当前目录的 `input/`。
3. 在当前目录运行对应脚本。
4. 结果会统一写入 `output/`。

本项目已经统一约定：

- 不通过 CLI 交互式传参
- 路径尽量写相对路径
- 外部软件优先从 `dependencies/bin/`、指定 conda 环境、系统 `PATH` 查找

## 备注
- 会自动查找 `cd-hit-est`。
- 通常接在 Trinity 组装结果之后。
